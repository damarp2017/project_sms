Student Management System (SMS)

User :
1. Admin
2. Lecturer/Dosen
3. Student/Mahasiswa


Fitur :
1. Admin
  - Mengelola data user (Pengguna Aplikasi)
  - Mengelola data student/Mahasiswa
  - Melihat data Achievement (Penghargaan Mahasiswa)
  - Ganti profile
2. Lecturer
  - Print data Student
  - Melihat data Achievement
  - Ganti Profil
3. student
  - Melihat data Achievement mahasiswa lain
  - Mengelola data achievementnya sendiri
  - Ganti profil


Note : Baru tampilan saja.

Copy project ini ke htdocs anda.

Untuk melihat login -> alamat urlnya : 'localhost/sms'

Untuk masuk ke user admin -> alamat urlnya : 'localhost/sms/admin'

Untuk masuk ke user lecturer -> alamat urlnya : 'localhost/sms/lecturer'

Untuk masuk ke user student -> alamat urlnya : 'localhost/sms/student'
